# DyslexiEase

DyslexiEase is an accessibility-focused web application designed to make reading more comfortable for individuals with dyslexia and other reading difficulties.

The platform provides simple yet effective tools that help users customize their reading experience, improve readability, and reduce visual stress while reading digital content.

## Problem Statement

Many individuals experience difficulties when reading standard digital text due to font styles, color contrasts, and visual processing challenges.

DyslexiEase aims to create a more accessible reading environment by allowing users to personalize how text is displayed and consumed.

## Features

### 🔊 Text-to-Speech

Convert written text into spoken audio, helping users listen to content instead of reading it manually.

### 🔤 Dyslexia-Friendly Font Changer

Switch to fonts that are designed to improve readability and make text easier to distinguish.

### 🎨 Color Overlay

Apply customizable color overlays to the screen to reduce visual stress and create a more comfortable reading experience.

### 🪄 Interactive Magic Wand Cursor

A custom cursor that adds a unique and engaging touch to the user experience.

## How It Works

1. Enter or paste text into the application.
2. Choose a dyslexia-friendly font if needed.
3. Apply a color overlay for improved visual comfort.
4. Use the text-to-speech feature to listen to the content.
5. Adjust settings according to personal reading preferences.

## Project Motivation

DyslexiEase was created to promote accessibility and inclusive learning by providing tools that help users adapt digital content to their individual reading needs.

Rather than forcing users to adapt to a fixed reading environment, the platform allows the reading environment to adapt to the user.
